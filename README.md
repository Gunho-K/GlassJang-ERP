
# GlassJang-ERP
거래처 관리, 수주 관리, 재고 관리, 매출 관리가 가능한 영업 관리 시스템

# 팀원
|이름|기능 설명|
|:------:|:-------------------------------------:|
|김건호|재고관리 구현, DB 제작 및 공유|
|김성중|로그인 구현, 메인 페이지 구현 및 각 코드 취합|
|장유리|수주 관리 구현|
|정세희|거래처 관리 구현|
|김다혜|매출 관리 구현, 오더|
